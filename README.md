=========================================
Программа поиска ошибок во входном файле.

Введите в консоли:  
```sh
make
./main < input-file
```  

Вы можете использовать флаги `-t` или `--test` для ручного тестирования модулей программы:  
```sh
make test
```  
или  
```sh
make
./main < test.txt --test
```
